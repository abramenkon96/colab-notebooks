# 🚀 Colab Notebooks Portfolio

Этот репозиторий содержит коллекцию специализированных блокнотов для Google Colaboratory, оптимизированных для работы с AI-моделями и медиаданными.

## 📂 Структура проекта

*   **[_experiments/](./_experiments/)** — Черновики, тесты и экспериментальные функции. Здесь ведется основная разработка до стабилизации кода.
*   **[examples/](./examples/)** — Проверенные примеры использования для конечных пользователей.
*   **[transcript/](./transcript/)** — Главный инструмент для транскрибации аудио/видео с использованием Whisper и Gemini API.
    *   [audio_transcriber_gemini.ipynb](./transcript/audio_transcriber_gemini.ipynb) — Стабильная версия транскрибатора с AI-обработкой.
*   **[archive/](./transcript/archive/)** — Предыдущие версии и устаревшие наработки.

## 🛠️ Основные инструменты

### Universal Audio Transcriber (Whisper + Gemini)
Мощный инструмент для превращения лекций и видео в структурированные академические конспекты.
- **Whisper:** Использует `large-v3-turbo` для быстрой и точной транскрибации.
- **Gemini:** Автоматически расставляет пунктуацию, абзацы и создает подробный саммари лекции.
